# IntelliGrade AI – Academic Performance Prediction and Recommendation System

## Overview

IntelliGrade AI is an academic performance prediction and recommendation system developed as a semester project. The system aims to predict students' academic performance using machine learning techniques and provide personalized recommendations for improvement.

## Features

- Academic performance prediction
- GPA estimation
- Academic risk analysis
- Personalized recommendations
- Interactive user interface

## Technologies Used

- Python
- Flask
- Machine Learning
- HTML
- CSS
- JavaScript
